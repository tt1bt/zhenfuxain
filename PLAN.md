# 初始实现计划（短期）

## 目标
- 在本仓库中完成可运行的训练 → 测试 → 简单演示闭环，优先保证可复现性。

## 里程碑
1. 环境与数据校验（1 天）
   - 验证 `DL` conda 环境能运行 `train.py` 与 `test.py`（最小样本集）。
   - 验证数据目录 `data/<dataset>` 可读。
2. 单次可复现训练（2 天）
   - 选择一个小数据集（或子集）运行 `python train.py`，保存 checkpoint。目标：训练脚本可运行且不报错。
3. 基线评估（1 天）
   - 使用 `test.py` 计算 mAP 与 P@K，并生成 PR 曲线 CSV。确保与仓库内现有 CSV 一致性格式。
4. 演示与打包（1 天）
   - 启动 `app_clrs_retrieval.py` 演示接口，能加载训练得到的模型并返回检索结果。

## 风险与依赖
- 需 GPU（或调整为小批量 CPU 测试）。
- 数据集未准备好时，使用小型子集或合成样本代替用于流程验证。

## 验收标准
- `train.py` 与 `test.py` 在目标环境无错误运行。
- 生成 checkpoint 与 PR 曲线 CSV 文件。演示能加载模型并返回合理检索结果。
