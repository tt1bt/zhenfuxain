# Step 4：实现 Dataset Loader

创建文件：

```
dataset/patternnet_dataset.py
```

代码：

```python
import os
from PIL import Image
from torch.utils.data import Dataset


class PatternNetDataset(Dataset):

    def __init__(self, root, transform=None):

        self.root = root
        self.transform = transform

        self.samples = []
        self.labels = []

        classes = sorted(os.listdir(root))

        self.class_to_idx = {c: i for i, c in enumerate(classes)}

        for cls in classes:

            cls_path = os.path.join(root, cls)

            for img in os.listdir(cls_path):

                img_path = os.path.join(cls_path, img)

                self.samples.append(img_path)
                self.labels.append(self.class_to_idx[cls])

    def __len__(self):

        return len(self.samples)

    def __getitem__(self, idx):

        img = Image.open(self.samples[idx]).convert("RGB")

        if self.transform:
            img = self.transform(img)

        label = self.labels[idx]

        return img, label
```

论文 backbone 使用 **ResNet-34** 。

代码：

```python
import torch
import torch.nn as nn
import torchvision.models as models


class HashModel(nn.Module):

    def __init__(self, hash_bits=32, num_classes=38):

        super().__init__()

        backbone = models.resnet34(pretrained=True)

        self.features = nn.Sequential(
            *list(backbone.children())[:-1]
        )

        self.hash_layer = nn.Linear(512, hash_bits)

        self.classifier = nn.Linear(hash_bits, num_classes)

    def forward(self, x):

        f = self.features(x)

        f = f.view(f.size(0), -1)

        hash_code = torch.tanh(self.hash_layer(f))

        pred = self.classifier(hash_code)

        return hash_code, pred
```

```python
import torch
import torchvision.transforms as transforms
from torch.utils.data import DataLoader

from dataset.patternnet_dataset import PatternNetDataset
from models.hash_model import HashModel

device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

    transforms.ToTensor(),

    root="data/PatternNet",
    transform=transform
)

train_loader = DataLoader(
    dataset,
    batch_size=32,
    shuffle=True
)

model = HashModel(hash_bits=32).to(device)

criterion = torch.nn.CrossEntropyLoss()

optimizer = torch.optim.Adam(
    model.parameters(),
    lr=1e-4
)

for epoch in range(10):

    model.train()

    total_loss = 0

    for img, label in train_loader:

        img = img.to(device)
        label = label.to(device)

        hash_code, pred = model(img)

        loss = criterion(pred, label)

        optimizer.zero_grad()
        loss.backward()
        optimizer.step()

        total_loss += loss.item()

    print("Epoch:", epoch, "Loss:", total_loss)
```

# Step3：实现 Centripetal Hash Loss

论文核心思想其实很简单：

**同一类别的 hash code 要靠近一个中心。**

也就是：

```
同类样本 → hash code 靠近 class center
```

数学形式：

```
Loss = || hash_code - center[label] ||²
```

# 第二步：写 Centripetal Loss

在 `utils/centripetal_loss.py` 写入：

```python
import torch
import torch.nn as nn


class CentripetalLoss(nn.Module):

    def __init__(self, num_classes, hash_bits):

        super().__init__()

        # 初始化 class centers
        self.centers = nn.Parameter(
            torch.randn(num_classes, hash_bits)
        )

    def forward(self, hash_code, labels):

        centers_batch = self.centers[labels]

        loss = (hash_code - centers_batch).pow(2).mean()

        return loss
```

解释：

```
centers → 每个类别一个中心
hash_code → 网络输出
labels → 类别标签
```

Loss：

```
|| hash - center ||
```

# 一、IAM 的核心思想（先理解一下）

在长尾数据中：

```
Head class   800张
Tail class     8张
```

普通训练时：

```
Loss = 平均
```

结果：

```
head 类统治训练
tail 学不好
```

IAM做的事情：

```
tail class loss ↑
head class loss ↓
```

即：

```
Loss = weight(label) * loss
```

权重：

```
w = 1 / sqrt(class_frequency)
```

类别越少：

```
weight 越大
```

# 三、写 IAM Loss

写入代码：

```python
import torch
import torch.nn as nn
import numpy as np


class IAMLoss(nn.Module):

    def __init__(self, class_counts):

        super().__init__()

        counts = np.array(class_counts)

        weights = 1.0 / np.sqrt(counts)

        weights = weights / weights.sum() * len(weights)

        self.weights = torch.tensor(
            weights,
            dtype=torch.float
        )

    def forward(self, pred, label):

        weights = self.weights.to(pred.device)

        loss = nn.CrossEntropyLoss(
            weight=weights
        )(pred, label)

        return loss
```

下面把你现在最常用的参数含义再单独汇总一下。

**训练参数（`train.py`）**
- `--root`：数据集根目录，格式是 `root/类别名/图片`，默认 `data/NWPU-RESISC45`
- `--imb_factor`：长尾不平衡系数，越小越不平衡（如 `0.1 > 0.05 > 0.01`）
- `--hash_bits`：哈希码长度（常用 `16/32/64`）
- `--epochs`：训练轮数
- `--batch_size`：训练 batch 大小
- `--center_batch_size`：计算类别中心时的 batch 大小
- `--alpha`：分类损失权重，总损失是 `hash_loss + alpha * cls_loss`
- `--gamma`：向心损失超参数（控制类内聚合强度）
- `--lr`：学习率（Adam）
- `--seed`：随机种子（保证复现）
- `--weights_out`：训练后权重保存路径
- `--device`：设备，`auto/cuda/cuda:0/cpu`
- `--query_ratio`：每个类别划到 query 集的比例（其余用于 train/db）
- `--split_path`：划分文件路径（不存在会自动生成）
- `--cls_weighting`：类别加权方式：`none/sqrt_inv/class_balanced`
- `--cb_beta`：`class_balanced` 模式的 beta 参数（通常接近 1，如 `0.9999`）

**测试参数（`test.py`）**
- `--root`：评测数据目录（应与训练数据同源）
- `--imb_factor`：构造长尾评测集时用的系数（要与训练设定一致）
- `--hash_bits`：哈希位数（要和权重匹配）
- `--weights`：单次评测用的权重文件
- `--weights_template`：`paper_like` 批量评测时的权重命名模板
- `--batch_size`：评测 batch 大小
- `--query_ratio`：query 比例（应与训练 split 一致）
- `--seed`：随机种子
- `--tsne`：开启后导出 t-SNE CSV
- `--tsne_max`：t-SNE 最大采样数
- `--device`：评测设备
- `--split_path`：划分文件路径
- `--topk`：`0` 表示全库 mAP，`>0` 表示 mAP@topk
- `--paper_like`：按 IF=`0.1/0.05/0.01` + bits=`16/32/64` 批量评测
